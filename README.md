# منصة طلاب الذكاء الاصطناعي – جامعة أم القرى

موقع ثابت باللغة العربية يعرض خطة برنامج الذكاء الاصطناعي بجامعة أم القرى مع مسارات تخصصية ومختبر مشاريع وبيئة تشغيل محلية جاهزة.

## المتطلبات
- Node.js 18+ (لتشغيل أوامر التطوير والبناء)

## التشغيل محلياً
```bash
npm install   # لا توجد تبعيات إضافية، الأمر يحضّر npm فقط
npm run dev   # يشغّل خادم تطوير ثابت على المنفذ 5173
```
ثم افتح المتصفح على: http://localhost:5173

## إنشاء نسخة إنتاج
```bash
npm run build
```
سيتم إنشاء مجلد `dist/` بنسخة جاهزة للنشر.

## هيكلة المشروع
- `src/`: ملفات الواجهة (HTML/CSS/JS).
- `scripts/dev-server.js`: خادم تطوير بسيط بالاعتماد على Node.js.
- `scripts/build.js`: نسخ الملفات إلى مجلد `dist/`.
- قسم المساعد الذكي يفتح ChatGPT مباشرة مع موجه عربي مبدئي.
