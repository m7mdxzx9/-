المسارات التخصصية

أربع مسارات لبناء عمق معرفي

اختر المسار الأنسب، مع محتوى ومشاريع وتوصيات أدوات لكل مسار.

الذكاء الاصطناعي العام

أساسيات النمذجة والتعلم الآلي والتعلم العميق مع تطبيقات متعددة.

  • مصادر: fast.ai, Coursera ML
  • مشروع: تصنيف صور، نظام توصية
  • أدوات: NumPy, Scikit-learn, PyTorch

معالجة اللغة العربية

بناء نماذج لغوية عربية وتطبيقات المحادثة وتحليل النصوص.

  • مصادر: CAMeL Lab, HuggingFace
  • مشروع: بوت مراسلة، تصنيف مشاعر
  • أدوات: transformers, spaCy, clean-text

الرؤية الحاسوبية

تحليل الصور والفيديو ودمج الرؤية مع تطبيقات الواقع المعزز.

  • مصادر: CS231n, OpenCV docs
  • مشروع: كشف أجسام، تتبع حركة
  • أدوات: OpenCV, Ultralytics YOLO

الذكاء الاصطناعي السحابي

تشغيل النماذج على السحابة، الحاويات، خطوط النشر، والمراقبة.

  • مصادر: AWS ML Stack, GCP Vertex
  • مشروع: نشر نموذج كـ API
  • أدوات: Docker, FastAPI, Grafana

البيئة البرمجية

بيئة تطوير محلية موجهة للطلاب

شغّل الخادم المحلي، ابنِ نسخة الإنتاج، واستفد من هيكل المشروع المنظم للمهام الأكاديمية.

خادم تطوير محلي

استخدم npm run dev لتشغيل خادم ثابت سريع يعرض الموقع مع إعادة تحميل فورية للملفات.

خط بناء بسيط

أمر npm run build ينشئ مجلد dist الجاهز للنشر عبر أي خدمة استضافة.

هيكلة واضحة

جميع الملفات الثابتة تحت src/ مع فصل أنماط الواجهة عن النصوص.
