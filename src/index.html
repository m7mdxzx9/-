الذكاء الاصطناعي السحابي

البيئة البرمجية

بيئة تطوير محلية موجهة للطلاب

شغّل الخادم المحلي، ابنِ نسخة الإنتاج، واستفد من هيكل المشروع المنظم للمهام الأكاديمية.

خادم تطوير محلي

استخدم npm run dev لتشغيل خادم ثابت سريع يعرض الموقع مع إعادة تحميل فورية للملفات.

خط بناء بسيط

أمر npm run build ينشئ مجلد dist الجاهز للنشر عبر أي خدمة استضافة.

هيكلة واضحة

جميع الملفات الثابتة تحت src/ مع فصل أنماط الواجهة عن النصوص.

قوالب محتوى عربية

تشكيلة أقسام جاهزة للتخصيص: الخطة، المسارات، المشاريع، الموارد.
